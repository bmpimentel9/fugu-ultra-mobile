# Fugu Ultra Mobile Page

Página 100% mobile-first em PT-BR sobre a história do modelo Sakana Fugu Ultra e sua principal diferença frente aos modelos monolíticos.

## Fontes usadas

- Sakana Fugu: https://sakana.ai/fugu/
- Release oficial: https://sakana.ai/fugu-release/
- OpenRouter: https://openrouter.ai/sakana/fugu-ultra
- Corporate Info: https://sakana.ai/company-info/
- Technical report: https://arxiv.org/abs/2606.21228

## Rodar local

```bash
npm install
npm run build
npm run preview
```
